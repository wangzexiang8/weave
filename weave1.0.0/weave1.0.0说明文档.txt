weave编程语言1.0.0版本介绍：
weave由学习c++的python小鬼编写
weave基于python编写，保留了python语言的诸多特性，但在语法方面稍作修改。

所有文件请在编写完毕后(可用记事本编写)保存为".wv"格式,并存储在一个文件夹中,输入路径后系统会自动适配。
如果文件夹中有多个“.wv”文件系统将全部按序执行。
在weave1.0.0版本中支持：
    1.定义整型变量
    2.定义字符串型变量（暂时无法使用,weave1.0.1已修复）
    3.利用print函数输出字符串，变量值。
    4.进行变量的加，减，乘，除。
基本语法：
    利用print函数输出字符串：
        此时print用“<”结尾
        print 您要输出的字符串<
        如：print weave<
        输出：weave
    利用int定义一个整型变量：
        int 变量名=整型变量值
        如：int a=1
        (此过程将不会有输出。)
    利用str定义一个字符串变量：
        str 变量名=字符串
        如：str word=hi
        (此过程不会有输出，该函数暂时无法使用，系统将自动跳过。)
    利用print输出变量值并进行四则运算：
        此时print用“?”结尾
        print 变量间的运算?
        如：
        int a=1
        int b=2
        print a+b?
        输出：3
以上就是weave1.0.0版本的所有语法。
weave将会持续进行更新。
